# TRACKR - Projet Transverse


## Organisation du projet

- **`Code Java/Trackr/`** : Racine du projet Java.
  - **`src/`** : Code source de l'application (package `trackr`).
- **`Documents et Rendus/`** : Documentation et rapports du projet.

## Installation & Lancement

### Configuration Requise
- Java JDK 17+
- IntelliJ IDEA ou Eclipse

### Ouvrir dans IntelliJ IDEA
1. Allez dans **File > New > Project from Existing Sources...**.
2. Sélectionnez le dossier `Code Java/Trackr`.
3. Choisissez **Import project from external model** > **Eclipse**.
4. Suivez les étapes en laissant les options par défaut.

### Ouvrir dans Eclipse
1. Allez dans **File > Import...**.
2. Sélectionnez **General > Existing Projects into Workspace**.
3. Choisissez le dossier `Code Java/Trackr` comme *Root Directory*.
4. Cliquez sur **Finish**.

### Lancement
La classe principale (Main) est **`FactoryMedia.java`**.
