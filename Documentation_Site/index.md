# Bienvenue sur Trackr

Trackr est une application conçue pour simplifier la gestion de vos films et séries préférés. Ce portail centralise toute la documentation technique et les rendus du projet.

## Le Projet

Ce projet est réalisé dans le cadre de la SAE-DEV par l'équipe Trackr de l'IUT d'Orsay. L'objectif est de mettre en pratique les principes de la programmation orientée objet et du déploiement continu.

---

### Points forts

- **Gestion Intelligente** : Organisez vos médias par catégories et saisons de manière fluide.
- **Aspect Social** : Suivez d'autres utilisateurs et consultez leurs recommandations.
- **Automation** : Intégration continue GitLab pour une qualité de code optimale.
