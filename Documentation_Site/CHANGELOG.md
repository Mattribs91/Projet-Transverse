# Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

## [1.0.0] - 2026-03-26
### Ajouté
- Migration complète vers Maven.
- Configuration du pipeline GitLab CI/CD (Tests + Build JAR).
- Ajout de la couverture de code avec JaCoCo.
- Modèles de tickets (Issues) et de Merge Requests.
- Guide de contribution et Licence.
- Fichier `.editorconfig` pour l'uniformisation du code.

### Supprimé
- Anciens fichiers de configuration Eclipse/IntelliJ (.classpath, .project, .iml).
- Dossiers de code en double.
